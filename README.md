# HH calls frontend

## How to set up

### Creating react project

`npx create-react-app call-modal-demo`

`cd call-modal-demo`

### After Setting up React. 
### Put the files into the folder /src

App.js

App.css  

CallModal.js

ResultModal.js

### Start the project

`npm start`
